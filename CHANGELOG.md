# Changelog

All notable changes to Carry Trade Monitor are documented here.

## [1.0.0]

### Added
- G10 carry-trade opportunities table across 14 classic pairs.
- Live FX rates from frankfurter.dev with derived cross rates.
- 30-day annualized volatility and risk-adjusted carry scoring.
- Signal badges (Attractive / Neutral / Risky).
- Summary stats bar and per-pair detail panel with a 90-day chart.
- Carry-return simulator and an educational section.
- Light theme, responsive layout, loading and error states.
- Published to GitHub Pages.
