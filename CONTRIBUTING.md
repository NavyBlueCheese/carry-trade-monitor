# Contributing

Thanks for your interest in Carry Trade Monitor!

## Development

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build
npm run deploy   # publish to GitHub Pages
```

## Guidelines

- Keep the UI clean, minimal, and accessible to non-finance users.
- Match the existing code style (React hooks, Tailwind utility classes).
- All FX data comes from frankfurter.dev; avoid adding API keys or backends.
- Run `npm run build` before opening a pull request to confirm it compiles.

## Reporting issues

Open an issue describing the problem, steps to reproduce, and your browser.
